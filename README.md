# qrry

Dead simple URL to QR conversion.

```shell
# Example markdown document written in Obsidian
$ qrry.py ~/notes/techstuff/os/ostree.md

Written: ~/notes/techstuff/os/ostree-qr.md
QR codes: ~/notes/techstuff/os/ostree-qr-qrcodes/
```
